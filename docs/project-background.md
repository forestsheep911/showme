# 项目背景

## 问题来源

ShowMe 来自一个很具体的家庭作业场景。

小朋友做作业时，经常会遇到这些情况：

- 有些字不认识，需要大人写出来或指出来。
- 有些字知道读音，但不知道怎么写。
- 有些短句、短文需要抄写，家长手写效率低，也未必写得足够清楚。
- 家长可能已经很久不频繁手写汉字，临时写字容易犹豫、笔顺变形或字形不规范。
- 即使家长会写，也还要找纸、找笔、找合适的位置，整个过程打断作业节奏。

这些问题本身不复杂，但发生频率高，而且常常出现在孩子已经坐在桌前、需要马上继续写作业的时候。

## 为什么网页有价值

网页比手写有几个优势：

- 输入比手写更快，尤其是短文、词组和生字列表。
- 字形更稳定，孩子看到的是清楚的印刷体。
- 字号可以放大，适合孩子抄写时反复查看。
- 手机、平板、电脑都能打开，不依赖纸笔。
- 内容可以随时改，不需要重新写一张纸。

因此，ShowMe 的核心价值不是“做一个高级编辑器”，而是把家长临时写给孩子看的内容变成清楚、可读、可抄的屏幕展示。

## 目标用户

主要用户有两类：

- 家长：负责快速输入孩子需要看的字、词、句子或短文。
- 孩子：看着屏幕上的内容完成认字、写字、抄写或订正。

家长是操作用户，孩子是阅读和抄写用户。产品设计要同时照顾两边：家长需要快，孩子需要清楚。

## 使用环境

典型环境包括：

- 家庭书桌旁。
- 家长手机递给孩子看。
- 平板或电脑放在孩子面前。
- 临时辅导作业、订正错字、抄写短文。

这些环境的共同点是：时间短、目的明确、不能有太多操作负担。
