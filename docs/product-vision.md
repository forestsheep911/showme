# 项目愿景

## 一句话愿景

ShowMe 要成为家长辅导孩子写字和抄写时最快打开、最少打扰、最清楚可读的临时展示屏。

## 产品原则

### 立即可用

打开网页后应该马上能输入，不需要登录、不需要建文档、不需要选择模板。

### 让孩子看清楚

字要足够大，字形要清楚，颜色和背景要有足够对比度。任何设置都应该服务于“孩子能不能看清楚并照着写”。

### 减少家长负担

家长不是在排版，也不是在管理学习资料。家长只是想把一个字、一句话、一段短文快速展示出来。产品不应把简单任务变复杂。

### 保持专注

默认界面应该尽量安静，减少按钮、说明、装饰和无关信息。孩子抄写时，屏幕上最重要的东西应该永远是文字本身。

### 可逐步增强

后续可以加入历史记录、常用短语、全屏、分行辅助、拼音等能力，但这些增强不能破坏“打开就能写，写完就能看”的核心路径。

## 非目标

ShowMe 暂时不追求成为：

- 完整的学习管理系统。
- 作文批改工具。
- 长文编辑器。
- 电子白板。
- 复杂的课堂课件工具。

这些方向都可能有价值，但当前最重要的是把家庭作业中的临时展示场景做到顺手。

## 成功标准

一个版本是否成功，可以用这些问题判断：

- 家长能否在几秒内输入并展示内容？
- 孩子能否不用解释就看清楚要抄的字？
- 短句、词语、短文是否都能稳定展示？
- 设置是否不会干扰孩子抄写？
- 移动设备和桌面设备上是否都能自然使用？
