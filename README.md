# ShowMe

ShowMe 是一个给孩子抄写用的临时大字展示工具。

它的核心场景很朴素：孩子做作业时遇到不会认、不会写、需要抄写的字词或短文，家长不一定能立刻手写得清楚，也不一定手边有纸笔。ShowMe 让家长直接在网页里输入内容，用足够大的字体、清楚的字形和合适的颜色展示给孩子看，孩子照着屏幕抄写即可。

## 当前能力

- 输入文字后以大字号展示。
- 支持浅色和深色主题。
- 支持多档字号。
- 支持简体中文、繁体中文、日文、韩文等字体选项。
- 支持切换文字颜色。
- 支持本地原型版展示房间：大屏新建房间，手机用 6 位配对码控制展示内容。
- 可部署为静态网站，打开网页即可使用。

## 产品定位

ShowMe 不是完整的作文、笔记或学习管理系统。它首先是一个低摩擦的家庭作业辅助屏：

- 家长负责快速输入。
- 屏幕负责清楚展示。
- 孩子负责照着抄写和练习。

它应当尽量减少设置、登录、保存、复杂编辑等干扰，让“现在就把这个字给孩子看清楚”成为第一优先级。
